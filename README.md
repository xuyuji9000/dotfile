# dotfile
personal dotfile

## .vimrc

1. cd ~
2. make .dotfile
3. git clone https://github.com/xuyuji9000/dotfile .dotfile
4. ln -s .dotfile\.vimrc .vimrc 
5. git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
6. vim
7. :BundleInstall

## Usage

### change colorscheme

```
:colorscheme badwolf
```
