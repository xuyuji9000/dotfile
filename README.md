# dotfile
personal dotfile

## .vimrc

1. cd ~
2. git clone https://github.com/xuyuji9000/dotfile
3. ln -s .dotfile\.vimrc .vimrc 
4. git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
5. vim
6. :BundleInstall

## Usage

### change colorscheme

```
:colorscheme badwolf
```
